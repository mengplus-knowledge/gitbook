## 简介

git-repo 是一个多仓库管理工具，在安卓开发中广泛使用，相比较git-submodule 有更加灵活的管理

## win环境安装
将解压拿到的exe程序放入`C:\Windows\git-repo.exe`路径即可,在cmd 或 git bash 中测试可用性。
