# 第一个工程
通过示例工程了解rt-thread开发过程,最快速度上手rt-thread开发工作
## 本节目标
1. 学会env`scons --target=iar`对示例工程的创建编译与下载仿真
2. 学会使用env工具修改FinSH控制台串口
3. finSH控制台使用体验
4. 使用env工具`scons --dist`导出工程

 **开发环境**

1. iar_8.40
2. rtthread版本v5.1.0
3. env Version 2.0.0
4. vscode git
5. win10 x64
6. mcu air32f103
